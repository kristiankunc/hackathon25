# Spaceship Battels

A real-time multiplayer battle game.

Answer coding questions to earn points, build a spaceship (guns, shields, engines, etc) and fight other players in real-time space battles!

## Features

### Guns

- Sten
- Mortar
- Laser pointer

### Shields

- Moravský shield
- Český shield
- Slezský shield
- Slovenský shield

### Engines

 - Škoda engine
 - Steam engine
 - Unity engine
